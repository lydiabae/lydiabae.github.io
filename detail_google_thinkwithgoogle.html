Think with Google

Think with Google is the go-to website for the latest consumer trends and market research from Google. Although the site already existed, I started this project as if I was starting from scratch and completely reworked the UX and IA of the site to create a simpler and more natural journey for the user.

Visit

Goal #1

Working with exisitng content

The existing site had various type of content including YouTube videos, infographic, articles and interactive tools. Even though the site had rich content but the site structure wasn't simple for the users. Search and filtering wasn't user friendly, it requires least 2-3 clicks to filter and find specific articles I was looking for.

I want to improve the content structure in very simple and leaner way while a user can jump to other sections. I analysed content types and group them in similar categories.

Goal #2

Page layout

The existing website contains different types of module: including embedded YouTube videos, interactive tools and downloadable Google documents. One of the biggest challenge was making the page more simple and legible. Instead of adding all modules in multiple column layout, I set up the content priority of them and placed them in meaningful order. I focused on three main criteria: Contrast, Alignment, Whitespace.

Whitespace - Applying carefully considered margins and space to create consistent structure. This allows a visual relief (avoiding distractions) for the users from the interface elements(windows, menus, secondary elements) of the browser and webpage.

Scannable Text - Left justified text creates a solid rectangles and it allows user to scan a larger amount of content easily. The headings are normally centred for a symmetrical, it's grabbing a strong attention before a user decide to dive into details.

Contrast - Nothing attracts the eye of the reader like strong contrast and distinctive patterns. I create a styleguide that brings consistent and flexible page layout through entire website. Heading styles are available from Heading 1 to Heading 4, each heading style contains unique font size, line height and colour. This was also changed on each breakpoint while I was developing the site with front end development teams.

Goal #3

Material design

As always, working with Google is both inspiring and challenging. When working on this, material design guidelines had only been released for apps and so we were one of the first teams to adapt this to website design. When thinking about the design, I wanted to work with Google’s new material design approach. I used cards to display and promote content to work with the clean lines throughout the website. Also material design emphasis on simple and meaningful animations. The Think with Google site was then used as an example of how to do this during this transition period.
