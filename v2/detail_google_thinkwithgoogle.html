Think with Google

Think with Google is a one-stop shop for consumer trends, marketing insights and industry research. I was responsible for redesigning the current site and localizing this in 5 different markets. The current site has quite a discombobulated user experience, so I started this project with a deep-dive analysis into improving the site architecture and user journey on the site. With over 800 stories to transfer and translate, the backend development is not a quick task.
