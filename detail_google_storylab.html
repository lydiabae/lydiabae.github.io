GOOGLE STORYLAB

StoryLab is an internal Google website which functions as a central repository to store case studies for different products. The use of bold imagery highlights featured stories and the ability to drill down into categories makes finding a relevant story in an organised and useful way. I was responsible for creating the information architecture, wireframe and a user friendly interface by using the existing site's functionality and content.

Skills used

  • Responsive Web Design
  • Information Architecture
  • User Interface Design
  • Art Direction
