GOOGLE + STORYLAB

StoryLab is an internal Google website which functions as a central repository to store case studies for different products. The site is built and deployed on Google App Engine, enabling us to create a dynamic and feature rich experience. The use of bold imagery highlights featured stories and the ability to drill down categories makes finding a relevant story in an organised an useful way. I was responsible for creating information architecture, wireframe and user friendly interface by using existing site functionality and the content.
